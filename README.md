### üé´ Integrated Ticket Management
TEMS bridges the gap between users and administrators with a robust ticketing engine aligned with **ITIL v3 and v4** standards. Users can raise tickets directly through the app to communicate with system administrators or AI agents.

| Ticket Type | Definition |
| :--- | :--- |
| **Incident** | An unplanned interruption or reduction in service quality. *Something is broken.* |
| **Security Incident** | A confirmed or suspected breach of security policy (e.g., Phishing, Malware). |
| **Service Request** | A formal request for something to be provided (e.g., access, hardware, info). |
| **Problem** | The underlying cause of one or more Incidents. *The disease, not just the symptom.* |
| **Change Request** | Any modification, addition, or removal that could affect IT services. |
| **Alert / Event** | Machine-generated notifications from external tools like AWS CloudWatch or Zabbix. |
